# 🙏 Bhandara Collection Tracker
A simple web application to manage donation lists and total amounts for community events.

### ✨ Features:
* **Add Entries:** Log names and donation amounts easily.
* **Auto-Calculate:** Shows the total collection in real-time.
* **Data Table:** Organized view of all contributors.

### 🛠️ Technology:
* HTML (Tables & Forms)
* CSS (Clean UI)
* JavaScript (Arrays and Math logic)

This is a practical project to show how we can use coding to help in real-life community work.
